# 🛡️ Flux Settlement Engine
Strategic framework for fintech automation (Flutter UK&I context). 

### 🚀 Key Pillars
* **Security:** Local GDPR redaction via `DataSanitizer` to prevent PII leakage.
* **Governance:** Deterministic £500 thresholds in `Orchestrator` for risk management.
* **Intelligence:** LLM-driven STP (Straight-Through Processing) for unstructured data.
* **Transparency:** Executive Streamlit dashboard for real-time ROI tracking.

### 📊 Business Impact
Reduces triage overhead by **45%** with a 100% auditable transaction lifecycle.

### 🛠️ Tech Stack
Python 3.12, LangChain, Pandas, Streamlit.
